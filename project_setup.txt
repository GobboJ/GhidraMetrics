Java version: 11 or later
Ghidra version: 10.1.5
GhidraDev version: 2.1.5
Gradle version: 6.8 or later

1. Download release from [https://github.com/NationalSecurityAgency/ghidra/releases/tag/Ghidra_<ghidra version>_build]
2. sudo unzip <download-folder>/ghidra_<ghidra version>_PUBLIC_<date>.zip -d /opt/
3. sudo apt install openjdk-<java version>-jdk
4. /opt/ghidra_<ghidra version>_PUBLIC/ghidraRun
5. open Eclipse, Help > Install new software > Add... > Archive... > select file /opt/ghidra_<ghidra version>_PUBLIC/Extensions/Eclipse/GhidraDev/GhidraDev-<ghidradev version>.zip > Add > mark the checkbox > Next > Accept terms of the license agreement > Finish > mark the checkbox of unsigned content > Trust Selected
6. Eclipse restart. Consent to GhidraDev opening ports for communicationg with Ghidra.
[5. open Eclise, File > Import projects from File System or Archive > Directory... > select root directory of plugin > Open > Finish]
7. open Eclipse, File > Import > Existing Gradle Project > Next > Browse... > select root directory of plugin > Open > Finish
8. right click on GhidraMetrics project > GhidraDev > Link Ghidra... > click on plus button > Add... > select folder /opt/ghidra_<ghidra version>_PUBLIC > Open > Apply and Close > Next > Next > Finish. In case of Java conflicts, use Ghidra Java.
